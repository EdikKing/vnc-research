# 快速开始 · 5 分钟跑起来

> 假设你有一台 Linux 服务器(VPS/旧电脑/树莓派),有 root 权限,从 0 到看到 VNC 浏览器画面。

---

## Step 1:装依赖(1 分钟)

```bash
sudo apt update
sudo apt install -y x11vnc novnc websockify
```python

需要安装的 3 个包:
- **x11vnc**:把 X11 屏幕暴露为 VNC 协议
- **novnc**:浏览器访问 VNC 的 Web 客户端(HTML5)
- **websockify**:WebSocket 代理(novnc 需要)

可选:
```bash
sudo apt install -y x11-apps imagemagick   # xdpyinfo / import 等诊断工具
```

---

## Step 2:启动虚拟显示(20 秒)

**情况 A**:服务器没有 X server(纯命令行)

```bash
# 1920x1080 像素 24 位色 + 允许匿名连接
Xvfb :99 -screen 0 1920x1080x24 -ac +extension GLX +render -noreset &
```text

**情况 B**:服务器已经有 X server(比如桌面版 Ubuntu)

直接用当前的 X,跳过这一步。

---

## Step 3:启动 VNC + noVNC(10 秒)

```bash
# VNC server, 端口 5900, 无密码(内网用)
x11vnc -display :99 -forever -shared -rfbport 5900 -nopw &

# noVNC, Web 端口 6080
websockify --web=/usr/share/novnc 6080 localhost:5900 &
```

---

## Step 4:启动 chromium(带调试端口)(5 秒)

```bash
# 软链绕过某些工具的路径过滤(可选)
sudo mkdir -p /etc/x11vnc && \
sudo ln -sf /tmp/.X11-unix/X99 /etc/x11vnc/auth 2>/dev/null || true

# 启动 chromium,带 --remote-debugging-port=9222
DISPLAY=:99 \
  /usr/lib/chromium/chromium \
    --no-sandbox \
    --disable-dev-shm-usage \
    --disable-gpu \
    --no-first-run \
    --remote-debugging-port=9222 \
    --remote-debugging-address=127.0.0.1 \
    --user-data-dir=$HOME/.config/chromium \
    --window-size=1920,1080 \
    --window-position=0,0 \
    about:blank &
```text

**关键参数解释**:
- `--no-sandbox`: root 用户运行需要
- `--disable-dev-shm-usage`: 避免 /dev/shm 空间问题
- `--remote-debugging-port=9222`: **核心**,让 AI agent 可以连进来
- `--user-data-dir`: profile 位置(登录态保存在这里)

---

## Step 5:浏览器访问 VNC(2 秒)

在**你自己的电脑**浏览器(任意设备,只要能访问服务器)打开:

```
http://你的服务器IP:6080/vnc.html
```python

应该看到一个 **1920×1080 的灰色桌面** + **一个 chromium 窗口**。

---

## Step 6:让 AI Agent 连进来(30 秒)

告诉你的 AI agent(Claude Code / Codex / Hermes Agent 任意)运行:

```python
python3 << 'EOF'
from playwright.sync_api import sync_playwright

with sync_playwright() as p:
    browser = p.chromium.connect_over_cdp("http://你的服务器IP:9222")
    print("Connected!")
    print(f"Pages: {len(browser.contexts[0].pages)}")
    page = browser.contexts[0].pages[0]
    page.goto("https://www.baidu.com")
    print(f"Title: {page.title()}")
EOF
```

如果输出 "Connected!" + "百度一下,你就知道",说明**整套架构跑通**。

## 常见问题

### Q:看到 VNC 但黑屏?
- 检查 chromium 是否启动: `pgrep -af chromium | head`
- 检查 DISPLAY 是否对: `pgrep -af Xvfb`
- 重新连接 noVNC(刷新浏览器)

### Q:AI agent 连不上 9222?
- 检查 9222 是否监听: `curl http://127.0.0.1:9222/json/version`
- 如果 0.0.0.0 不监听,改 `--remote-debugging-address=0.0.0.0`(仅内网)

### Q:chromium 启动报 "Missing X server"?
- 加 `DISPLAY=:99` 前缀
- 如果 Xauthority 报错,见 docs/04-踩坑记录.md

### Q:登录小红书/抖音被 datacenter IP 拦?
- 没法解决(本项目不走 stealth 路线)
- **但你之前登录过的账号,cookie 还在,agent 用登录态正常访问**
